# LinkTree
Building a Linktree-style page in React with one JavaScript feature that brings it to life, then deploying it to GitHub Pages.

This is my link tree that I built using react with vite. I added a couple javascript features such as dark mode, a greating on top of the page depending what time of the day it is, and a link counter counting the amout of times a link was counted. I would have liked to add a comment or feedback feature to make the link tree more useful but maybe with more time I could have done it. The feature I picked I feel give the link tree give some interactive feeling such as switching to dark or light mode or seeing the amount of clicks a link has.
